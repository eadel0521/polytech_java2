# 2026-06-29 Generic / Enum

## 오늘 배운 내용
- Generic을 활용해 타입에 유연한 클래스 만들기
- Enum으로 고정된 상수 집합 정의하기
- Generic + Enum을 함께 활용하는 방법 (StrongBox 과제)

## 기억할 것
- Generic의 타입 파라미터 <T>는 인스턴스 생성 시 결정됨
- Generic을 쓰면 get() 시 별도 타입 캐스팅이 불필요함
- Enum은 고정된 상수 집합을 타입 안전하게 표현할 때 사용
- 전통적인 Enum은 상수만 나열, 상수별 데이터가 필요하면 향상된 Enum 사용

## 어려웠던 점
- Enum의 상수별 한도(limit)를 어디에 저장할지 헷갈렸음
- 전통적 Enum으로 구현하면 StrongBox 내부에 if-else가 길어지는 문제 발생

## 해결 방법
- 열쇠별 한도는 KeyType이 책임져야 할 데이터임을 인식
- 필드와 생성자, 메서드가 전부 포함된 '향상된 Enum'을 사용하면 새로운 key를 
추가할 때마다 get()코드를 수정할 필요가 없음

## 내일 더 공부할 것
- 향상된 Enum의 필드, 생성자, 메서드 구조를 더 깊이 이해하기
- Enum에 추상메서드를 정의하고 각 상수별로 다르게 구현하는 방법 공부할 예정
