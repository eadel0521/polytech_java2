# 2026-06-22 추상매서드 / 추상클래스

## 오늘 배운 내용
- 추상클래스와 추상매서드를 활용하는 경우
- 추상클래스의 상속
- 인터페이스 구현과 활용

## 기억할 것
- 추상메서드가 하나라도 포함된 클래스 == 추상클래스
- 추상클래스는 'new' 로 객체생성이 안됨 (자식 클래스 생성만 가능)
- 인터페이스의 모든 메서드는 추상메서드 (추상메서드만 가지는 추상클래스 == 인터페이스)
- 코드 작성할 때 캡슐화하고 생성자 까먹지 말 것

## 어려웠던 점
- 'Computer' 의 생성자를 만들고나서 test코드 오류 발생

## 해결 방법
- 생성자를 만들면서 인수를 넣었으면 test의 Computer도 동일하게 수정해야했는데 깜빡했음

## 내일 더 공부할 것

다형성이라는 개념을 알게되었는데 이 부분에 대해서 좀더 공부할 예정이다.
